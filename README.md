# 🔐 Vault9 – Password & Secret Locker

> A modern, offline-first Android password manager and secret locker built with Kotlin and Jetpack Compose.

---

## ✨ Features

- 🔒 Encrypted local vault
- 🔑 Password manager
- 📝 Secure notes
- 💳 Credit & debit card storage
- 📄 Secure document storage
- 🔐 PIN & biometric authentication
- 🔍 Fast search
- ⭐ Favorites
- 🌙 Light & Dark mode
- 🎨 Material You UI
- 📱 Offline-first
- 🚫 No ads, No tracking

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
- Room Database
- SQLCipher
- Android Keystore
- Kotlin Coroutines
- Hilt

## 🚀 Getting Started

### Requirements

- Android Studio (Latest)
- Android SDK 29+
- Kotlin 2.x

### Installation

```bash
git clone https://github.com/USERNAME/VaultX.git
```

Open the project in Android Studio and run it.
